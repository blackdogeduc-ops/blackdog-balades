# Black Dog Balades – Version simple (auth uniquement)

Ce dossier contient une version **très simple** de l'application :

- `index.html` : interface de connexion / création de compte
- `style.css` : style noir & orange
- `firebase-config.js` : configuration Firebase (à remplir)
- `script.js` : logique de base (login + register)

## 1. Remplir firebase-config.js

Va dans la console Firebase → Paramètres du projet → Config SDK Web, copie le bloc `const firebaseConfig = { ... }` et remplace **uniquement** les valeurs `A_REMPLACER` dans `firebase-config.js`.

Ne change pas les mots-clés (apiKey, authDomain, projectId, etc.).

## 2. Héberger sur GitHub Pages

Envoie tous les fichiers dans un dépôt GitHub, puis active GitHub Pages.

L'écran te permettra ensuite de créer un compte et de te connecter avec Firebase Auth.
